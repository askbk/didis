# Reredis

Reredis is a reimplementation of Redis for learning/fun.

Progress:

- [x] Key-value operations (`DEL`, `EXISTS`)
- [x] Strings (`GET`, `SET`, `INCR`)
- [x] Lists (`LPUSH`, `LPOP`, `RPUSH`, `RPOP`, `LLEN`, `LMOVE`)
- [ ] Sets
- [ ] Wider characters (UTF8?)
- [ ] Hashes
- [ ] Sorted sets
- [ ] HyperLogLog
- [ ] Client-server architecture
